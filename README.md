# Coding Adventure: Reaction Diffusion Patterns

## History of the Project

This section provides a record of how this project came to fruition from the initial idea, to the research, to the final implementation.

After watching Sebastian Lague on youtube for years, I wanted to make my own attempt at a coding adventure. I had a few ideas for projects, but I wanted to do something visually intriguing, and technically challenging. One idea that I had found interesting was reaction diffusion patterns. Note that when I started this project, I had no idea what reaction diffusion patterns were or even how to handle intense computations with gpus. But that was the point! So I started with the following resources:

- [Karl Sim's Reaction-Diffusion Tutorial](https://karlsims.com/rd.html)
- [Reaction-Diffusion Simulation using the Gray-Scott Model](https://pmneila.github.io/jsexp/grayscott)
- [Reaction-Diffusion Course (Amazing Resource!)](https://biologicalmodeling.org/)

These resources provided a great starting point, explaining the history of reaction diffusion patterns, the mathematical models behind them, how to implement them in code, and the resulting simulations. For practical reasons, I decided to implement the Gray-Scott model. Once that decision was made, I had to figure out how to implement the model in code. For this I used this tutorial on [WebGPU by Google](https://codelabs.developers.google.com/your-first-webgpu-app).
